# Como executar o projeto

## Back end
Pré-requisitos: 
- Node
- Yarn
- Mongo DB

## Antes de Executar
Preencher o arquivo .env:
- MONGO_URI
- MONGODB_USER
- MONGODB_ADM_PASS

## Executar
No terminal executar os seguintes comandos:
- yarn install
- yarn dev

## Testes
Confira a collection do Postman:
- Bis2Bis.postman_collection.json
